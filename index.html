CodeVault

CodeVault is a lightweight macOS backup utility for developers and power users.

The app creates local ZIP backups from selected source folders and saves them to a destination folder chosen by the user. It is especially useful for backing up Xcode, Swift, Visual Studio Code, Python, JavaScript, TypeScript, web, and Git-based projects.

Features

  • Back up one or more source folders
  • Create ZIP archives locally on your Mac
  • Save backups to cloud-synced folders such as iCloud Drive, OneDrive, or Google Drive
  • Include or exclude hidden files
  • View backup progress and activity
  • Use German, English, or system language

CodeVault does not upload files by itself and does not collect user data.
